# Urdu Voiceover Lines Only

## Video 1: Service dhoondhna aur provider select karna
1. Assalamualaikum. GharKaam par aap ghar ke kaam ke liye trusted service asani se dhoondh sakte hain.
2. Search box mein apni zarurat likhein, jaise plumber, electrician ya cleaning.
3. Phir location aur category select karke search button dabayein.
4. Results mein provider ka area, price aur reviews compare karein.
5. Jo service pasand aaye us ki detail page open karke behtar faisla karein.
6. Agar aap ko madad chahiye ho to WhatsApp support se bhi raabta kar sakte hain.

## Video 2: Account banana aur booking submit karna
1. Booking karne ke liye sab se pehle apna account bana lein ya login karein.
2. Register form mein basic details fill karke account create karein.
3. Agar account pehle se ho to seedha login kar lein.
4. Login ke baad service detail page par Book button dabayein.
5. Booking form mein date, address aur notes add karein.
6. Submit karte hi aap ki booking request system mein save ho jayegi.
7. Dashboard se aap apni booking status asani se dekh sakte hain.

## Video 3: Provider dashboard aur service create karna
1. Provider login ke baad aap apna poora dashboard use kar sakte hain.
2. Dashboard se aap apni services, bookings aur profile sab manage kar sakte hain.
3. Profile complete karein taake customer ko aap ki information clear nazar aaye.
4. Add Service page open karke nayi service create karein.
5. AI Service Builder mein simple prompt likh kar quick draft hasil karein.
6. System title, description, pricing aur draft image ready kar dega.
7. Review ke baad service save karein aur customers ke liye live ho jaayein.
