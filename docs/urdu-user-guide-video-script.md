# Urdu User Guide Video Script

## Video 1: Service dhoondhna aur provider select karna
Duration target: 60 to 70 seconds
Audience: New customers

Voiceover:
Assalamualaikum. GharKaam par aap asani se apne ghar ke liye service dhoondh sakte hain. Sab se pehle homepage par aayein. Search box mein apni zarurat likhein, jaise plumber, electrician ya cleaning. Phir apni location select karein aur category choose karein. Search button dabate hi aap ke samne relevant services aur providers aa jayenge. Har card par aap provider ka naam, area, price aur reviews dekh sakte hain. Jo service pasand aaye us par click karke detail page open karein. Isi tarah aap asani se best provider compare karke select kar sakte hain.

On-screen flow:
1. Homepage open
2. Search field fill
3. Location and category select
4. Results list open
5. Service detail page open

## Video 2: Account banana aur booking submit karna
Duration target: 70 to 80 seconds
Audience: Customers

Voiceover:
Agar aap booking karna chahte hain to pehle apna account bana lein. Login ya Register button par click karein. Apna naam, email, phone aur password enter karke account create karein. Login hone ke baad kisi bhi service detail page par jaakar Book button dabayein. Booking form mein apni preferred date, address aur short notes likhein. Submit karne ke baad aap ki booking request system mein save ho jayegi. Is ke baad aap dashboard se apni booking status bhi dekh sakte hain.

On-screen flow:
1. Register page
2. Login page
3. Service detail page
4. Booking form submit
5. Dashboard booking status

## Video 3: Provider dashboard aur service create karna
Duration target: 80 to 90 seconds
Audience: Providers

Voiceover:
Agar aap provider hain to login ke baad dashboard open karein. Yahan se aap apni profile, bookings aur services manage kar sakte hain. Pehle profile complete karein, service area aur availability update karein. Us ke baad new service create karein. Aap simple prompt likh kar AI Service Builder bhi use kar sakte hain, jaise mere pass plumber aur electrical service hai. System title, description aur pricing draft banane mein help karega. Review karke service save karein aur phir aap ki service customers ko show hone lagegi.

On-screen flow:
1. Provider login
2. Dashboard open
3. Profile update
4. Service form or AI builder
5. Save service
